You can import expressions using CSV and JSON. For importing expressions there are two options:

1. Importing expressions for a category

  • At first you have to select the category for which to import the expressions
  • Then you need to select the file to import expressions from
  • The expressions are now imported
Note: If an expression already exists in this category, it is not created again and will not produce a warning/error.

The files have to have the following format:

*.csv 
expression1
expression2
.
.
.

Example file (.csv)

*.json 
[ "expression1", "expression2", ... ]

Example file (.json)

2. Importing expressions for multiple categories

  • At first you need to select the file to import from
  • Then the expressions and categories are importet
Note: If an expression (in this category)/category already exists, it is not created again and will not produce a warning/error.

The files have to have the following format:

*.csv 
category1;expression1_1
category1;expression1_2
.
.
.
category2;expression2_1
category2;expression2_2
.
.
.

Example file (.csv)

*.json 
[
{{ '{' }}
"category": "category1",
"expressions": [ "expression1_1", "expression1_2", ... ]
},
{{ '{' }}
"category": "category2",
"expressions": [ "expression2_1", "expression2_2", ... ]
},
.
.
.
]

Example file (.json)
